# 8051 Microcontroller Projects Embedded Suite ⚙️⚡

Welcome to my repository dedicated to practical hardware applications using the **8051 Microcontroller architecture**. This repository serves as a portfolio of embedded systems projects, focusing on hardware-software interfacing, peripheral control, and digital-to-analog signal synthesis written in **C / Assembly language**.

---

## 📂 Repository Structure

The projects are organized into modular folders, each containing its complete source code, custom headers, and configuration files:

### 1. 🏎️ [Motor_control](./Motor_control)
* **Description:** A system designed to manage and control DC motors using the 8051 microcontroller. 
* **Key Features:** Implements directional controls (Clockwise/Counter-Clockwise) and safe stopping sequences via low-level register manipulation.

### 2. 🌊 [DAC_converter](./DAC_converter)
* **Description:** An advanced digital-to-analog synthesis application aimed at generating smooth periodic signals.
* **Key Features:** Utilizes a **Look-up Table (LUT)** mapping dynamic digital values to Port 2 paired with **Timer 0 in Mode 2 (8-bit Auto-Reload)** for flawless high-frequency Sine Wave generation.

---

## 🛠️ Tech Stack & Tools

* **Core Architecture:** Intel 8051 Microcontroller (AT89C51 / AT89C52)
* **Programming Languages:** Embedded C, Keil C51 Compiler
* **Development Environments:** Keil uVision IDE
* **Simulation & Testing:** Proteus Design Suite (for schematic capture and oscilloscope signal analysis)
